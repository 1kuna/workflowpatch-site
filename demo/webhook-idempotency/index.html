A small mock workflow that checks inbound events before any downstream write. Safe rows become an accepted event ledger; duplicate ids, invalid signatures, unsupported event types, tenant mismatches, inactive tenants, and missing fields are blocked visibly. No client data, credentials, APIs, or live writes are used.
