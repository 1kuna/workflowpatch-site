Reverse logistics routing demo output.

A small mock workflow that routes return and overstock events against brand guardrails before any warehouse, marketplace, pricing, or brand-partner action happens. Safe rows become a route review queue; duplicates, missing policies, and blocked channels become exceptions. No client data, credentials, or live inventory actions are used.

Seven mock return and overstock events plus three mock brand guardrail rows.

Accepted

Three events become route-review rows for resale or repair. Every route still needs human approval.

Failure Path

Duplicate events, missing brand policies, and incomplete warehouse status are blocked with next steps.
